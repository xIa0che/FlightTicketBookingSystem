# FlightTicketBookingSystem
A simple java application of booking tickets, which uses swing and oracle with a three-tier architecture.

###Function:
 
1. 用户注册登录，查看修改用户信息
2. 机票查询，预订
3. 退票，改签同一航空公司其他日期的航班
4. 管理员添加修改删除航班
